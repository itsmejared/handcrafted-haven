# Handcrafted Haven

A virtual marketplace connecting artisans and crafters
with customers who appreciate unique handmade products.

## Project Description

Handcrafted Haven provides a platform for artisans to
showcase and sell their unique handcrafted items. It
connects talented creators with customers who appreciate
the beauty and quality of handmade products.

## Tech Stack

- Front-End: Next.js, React, Tailwind CSS
- Back-End: Node.js, PostgreSQL
- Deployment: Vercel
- Project Management: GitHub Projects

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000
